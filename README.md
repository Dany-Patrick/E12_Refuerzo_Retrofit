# E12_Refuerzo_Retrofit
  # Desarrollo
  
1. Este proyecto contiene una base para desarrollar una APP como [Unsplashy](https://play.google.com/store/apps/details?id=com.desafiolatam.unsplashy),
   la cual realiza un request a unsplash.com y muestra 10 imagenes aleatorias, cada vez que se abre la aplicacion.
   
   # Instrucciones
   
1.  Debes hacer `git clone` al repositorio y abrirlo en Android Studio.
2.  Debes añadir los  modelos que correspondan.
3.  Debes añadir retrofit y en una Asynctask solicitar el resultado del request.
4.  Para ayudarte a crear tu modelo, puedes utilizar la siguiente página.
